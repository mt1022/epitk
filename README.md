# epitk

**Epi-Transcriptome Kit** — a Python toolkit for analysis of epitranscriptomic data.

[![PyPI - Version](https://img.shields.io/pypi/v/epitk)](https://pypi.org/project/epitk/)
[![PyPI - Python Version](https://img.shields.io/pypi/pyversions/epitk)](https://pypi.org/project/epitk/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![CI](https://github.com/mt1022/epitk/actions/workflows/ci.yml/badge.svg)](https://github.com/mt1022/epitk/actions/workflows/ci.yml)

## Installation

```bash
pip install epitk
```

## Usage

```python
import epitk

print(epitk.__version__)
```

## Development

Clone the repository and install in editable mode with the development extras:

```bash
git clone https://github.com/mt1022/epitk.git
cd epitk
pip install -e ".[dev]"
```

Run the test suite:

```bash
pytest
```
